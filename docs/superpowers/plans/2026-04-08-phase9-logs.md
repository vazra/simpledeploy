# Phase 9: Log Streaming - Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stream container logs to clients via WebSocket. Support follow mode (live), historical tail, service filtering. No log persistence (Docker handles retention).

**Architecture:** Client opens WebSocket to `/api/apps/{slug}/logs`. Server attaches to Docker container log stream via `ContainerLogs` API. Multiplexes stdout/stderr into JSON messages. Each WebSocket client gets its own Docker stream, cancelled when WebSocket closes.

**Tech Stack:** gorilla/websocket, Docker ContainerLogs API, existing docker/api packages

#### WebSocket Handler (logs.go):

```go
var upgrader = websocket.Upgrader{
    CheckOrigin: func(r *http.Request) bool { return true },
}

func (s *Server) handleLogs(w http.ResponseWriter, r *http.Request) {
    slug := r.PathValue("slug")

    // verify app exists
    app, err := s.store.GetAppBySlug(slug)
    // ...

    // parse query params
    follow := r.URL.Query().Get("follow") != "false"  // default true
    tail := r.URL.Query().Get("tail")
    if tail == "" { tail = "100" }
    since := r.URL.Query().Get("since")
    service := r.URL.Query().Get("service")

    // find container(s) for this app
    containerName := fmt.Sprintf("simpledeploy-%s-%s", slug, service)
    // if no service specified, find the first container with simpledeploy.project=slug

    // upgrade to WebSocket
    conn, err := upgrader.Upgrade(w, r, nil)
    // ...
    defer conn.Close()

    // attach to Docker logs
    ctx, cancel := context.WithCancel(r.Context())
    defer cancel()

    // cancel on WebSocket close
    go func() {
        for {
            if _, _, err := conn.ReadMessage(); err != nil {
                cancel()
                return
            }
        }
    }()

    logOpts := container.LogsOptions{
        ShowStdout: true,
        ShowStderr: true,
        Follow:     follow,
        Tail:       tail,
        Timestamps: true,
    }
    if since != "" {
        logOpts.Since = since
    }

    reader, err := s.docker.ContainerLogs(ctx, containerName, logOpts)
    // ...
    defer reader.Close()

    // Docker log stream has 8-byte header per frame:
    // [0] stream type (1=stdout, 2=stderr)
    // [1-3] padding
    // [4-7] uint32 frame size (big endian)
    // then frame_size bytes of log data

    hdr := make([]byte, 8)
    for {
        _, err := io.ReadFull(reader, hdr)
        if err != nil { break }

        streamType := "stdout"
        if hdr[0] == 2 { streamType = "stderr" }

        size := binary.BigEndian.Uint32(hdr[4:8])
        line := make([]byte, size)
        _, err = io.ReadFull(reader, line)
        if err != nil { break }

        // parse timestamp from line if present
        msg := map[string]string{
            "stream": streamType,
            "line":   strings.TrimRight(string(line), "\n"),
        }

        // extract timestamp if line starts with RFC3339
        lineStr := string(line)
        if idx := strings.Index(lineStr, " "); idx > 20 {
            msg["ts"] = lineStr[:idx]
            msg["line"] = strings.TrimRight(lineStr[idx+1:], "\n")
        }

        if err := conn.WriteJSON(msg); err != nil { break }
    }
}
```

Actually simpler: CLI can just call Docker logs directly without going through the API:
```go
func runLogs(cmd *cobra.Command, args []string) error {
    appName := args[0]
    follow, _ := cmd.Flags().GetBool("follow")
    tail, _ := cmd.Flags().GetString("tail")
    service, _ := cmd.Flags().GetString("service")

    dc, _ := docker.NewClient()
    defer dc.Close()

    containerName := fmt.Sprintf("simpledeploy-%s-%s", appName, service)
    // if no service, use appName as-is and find first matching container

    reader, _ := dc.ContainerLogs(ctx, containerName, container.LogsOptions{
        ShowStdout: true, ShowStderr: true,
        Follow: follow, Tail: tail, Timestamps: true,
    })
    defer reader.Close()

    // parse docker log stream and print to stdout
    // same 8-byte header parsing as WebSocket handler
}
```
