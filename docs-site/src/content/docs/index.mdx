## Why SimpleDeploy

<CardGrid stagger>
  <Card title="One Binary" icon="rocket">
    Single Go binary. No runtime, no Kubernetes, no agent to babysit. Drop it on a VPS and go.
  </Card>
  <Card title="Automatic HTTPS" icon="approve-check">
    Embedded Caddy handles Let's Encrypt, custom certs, and routing. Zero config for the happy path.
  </Card>
  <Card title="Built-in Backups" icon="seti:db">
    Postgres, MySQL, Mongo, Redis, SQLite, volumes. Local or S3. Schedules and retention included.
  </Card>
  <Card title="Metrics + Alerts" icon="bars">
    Per-service CPU, memory, request rates, latency, errors. Webhook alerts on rules you define.
  </Card>
  <Card title="Remote CLI" icon="laptop">
    kubectl-style contexts. Pull, diff, apply compose files against any server you have a key for.
  </Card>
</CardGrid>

## How it works

<Steps>

1. **Install the binary on your server.**

   ```bash
   curl -fsSL https://github.com/vazra/simpledeploy/releases/latest/download/install.sh | sh
   ```

2. **Add `simpledeploy.*` labels to your `docker-compose.yml`.**

   ```yaml
   services:
     web:
       image: nginx:alpine
       labels:
         simpledeploy.app: my-site
         simpledeploy.endpoints: "my-site.example.com -> 80"
   ```

3. **Apply.**

   ```bash
   simpledeploy apply ./docker-compose.yml
   ```

</Steps>

## Install

<Tabs syncKey="install">
  <TabItem label="macOS (Homebrew)">
    ```bash
    brew install vazra/tap/simpledeploy
    ```
  </TabItem>
  <TabItem label="Ubuntu / Debian (apt)">
    ```bash
    curl -fsSL https://pkg.simpledeploy.dev/apt/setup.sh | sudo sh
    sudo apt install simpledeploy
    ```
  </TabItem>
  <TabItem label="Docker (any distro)">
    ```bash
    sudo mkdir -p /etc/simpledeploy /var/lib/simpledeploy
    curl -fsSL https://raw.githubusercontent.com/vazra/simpledeploy/main/deploy/docker-compose.example.yml \
      | sudo tee /etc/simpledeploy/docker-compose.yml
    cd /etc/simpledeploy && sudo docker compose up -d
    ```
  </TabItem>
  <TabItem label="Linux binary (curl)">
    ```bash
    curl -fsSL https://github.com/vazra/simpledeploy/releases/latest/download/install.sh | sh
    ```
  </TabItem>
</Tabs>
