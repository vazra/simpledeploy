SimpleDeploy is a small open-source project. The fastest way to get help depends on what you need.

## Ask a question

Open a thread in [GitHub Discussions](https://github.com/vazra/simpledeploy/discussions). Tag with `question`. Search first; chances are someone hit the same thing.

## Report a bug

Open an issue in [GitHub Issues](https://github.com/vazra/simpledeploy/issues). See [Reporting a bug](/community/reporting-bug/) for what to include.

## Report a security vulnerability

Do NOT open a public issue. Email security@vazra.example. See [Reporting a vulnerability](/community/reporting-vuln/).

## Real-time chat

Coming soon: a community chat on Discord/Matrix. Watch the [README](https://github.com/vazra/simpledeploy) for the link.

## Stack Overflow

Tag your question `simpledeploy`. We watch the tag.

## Commercial support

None at present. If you need a service-level commitment for a deployment, reach out via Discussions and we will connect you with maintainers who consult.

## Help improve the docs

Every page has an "Edit this page" link in the footer. Typos and clarifications are very welcome.
