SimpleDeploy adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct.

The full text lives at the repo root in [`CODE_OF_CONDUCT.md`](https://github.com/vazra/simpledeploy/blob/main/CODE_OF_CONDUCT.md).

To report unacceptable behavior, contact the maintainers via the channels listed in [Support](/community/support/).
